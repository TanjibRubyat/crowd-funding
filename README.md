# Al Moon International — Landing Page

A responsive landing page for M/S Al Moon International, a government-approved overseas manpower agency based in Dhaka, Bangladesh (License No. RL-1854).

## What the site covers

- Company introduction and services overview
- Government approval and licensing information
- Middle East placement specialisation (Saudi Arabia, UAE, Qatar, Gulf countries)
- Leadership profile
- Contact section with call-to-action

## Live site

https://almooninternational.com/

## Tech stack

| Component | Technology |
|-----------|-----------|
| Framework | React (Vite) |
| Styling | Tailwind CSS |
| Language | JavaScript |

## Getting started

```bash
git clone https://github.com/TanjibRubyat/AlMoon-Landing.git
cd AlMoon-Landing

npm install

npm run dev
```

App runs at `http://localhost:5173`

To build for production:

```bash
npm run build
```
